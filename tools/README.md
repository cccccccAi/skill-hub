# 预置依赖工具

本目录存放 cc-switch 安装包（v3.10.3），供用户直接使用，无需另外下载。

## 目录结构

```
tools/
├── macos/
│   └── CC-Switch-v3.10.3-macOS.zip       # macOS 版（Finder 双击解压）
└── windows/
    └── CC-Switch-v3.10.3-Windows.msi     # Windows 版
```

## 更新方式

从 https://github.com/farion1231/cc-switch/releases 下载最新版本，替换对应文件并更新文件名。
