# 预置依赖工具

本目录存放安装所需的工具包，供用户直接使用，无需另外下载。

## 目录结构

```
tools/
├── macos/
│   └── CC-Switch-v3.10.3-macOS.zip             # cc-switch macOS 版（Finder 双击解压）
└── windows/
    ├── CC-Switch-v3.10.3-Windows.msi           # cc-switch Windows 版
    ├── WindowsTerminal-v1.23-x64.zip           # Windows Terminal（64位）
    └── WindowsTerminal-v1.23-x86.zip           # Windows Terminal（32位）
```

## 更新方式

- **cc-switch**：从 https://github.com/farion1231/cc-switch/releases 下载最新版本
- **Windows Terminal**：从 https://github.com/microsoft/terminal/releases 下载最新版本
