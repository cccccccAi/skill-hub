- 注册学生数：6,200,000
- 月活跃学生数：4,500,000（MAU）
- 周活跃学生数：2,800,000（WAU）
- 日活跃学生数：850,000（DAU）
- DAU/MAU 比率：0.19（低频产品，正常范围）

## 留存分析（关键发现）

### 按功能分组的 D30 留存率

| 用户分组 | D30 留存 | 样本量 |
|---------|---------|--------|
| 使用错题本的学生 | **85%** | 1,890,000 |
| 未使用错题本的学生 | **51%** | 2,310,000 |
| 使用知识点诊断的学生 | 78% | 2,604,000 |
| 使用智能推荐的学生 | 82% | 966,000 |
| 仅查看成绩的学生 | 48% | 540,000 |

**核心发现：错题本是最强的留存预测因子。使用错题本的学生 D30 留存率(85%)比不使用的(51%)高出 34 个百分点。**

趋势：新用户留存逐月改善，说明产品体验在持续优化。

## 考试周期行为

| 时期 | DAU 倍率 | 说明 |
|------|---------|------|
| 普通周 | 1.0x | 基准 |
| 月考前一周 | 1.5x | 错题本和练习使用量增加 |
| 月考后三天 | 2.5x | 查成绩 + 查分析高峰 |
| 期中考试周 | 3.0x | 全年最高峰之一 |
| 期末考试周 | 3.2x | 全年最高峰 |
| 寒暑假 | 0.3x | 大幅下降，仅少量学生使用错题本复习 |
