# 第3关：教育科技竞品分析

> 注意：智学网是科大讯飞旗下产品，讯飞不列为竞品。

## 目标
用 competitive-analysis 框架产出一份竞品简报，包含竞品地图、功能矩阵、定位分析和战略建议。

## 时长
约 12 分钟

## 教学流程

### Step 1：真实数据入口

告诉用户：「这一关做竞品分析。你可以补充自己掌握的竞品情报，让分析更有针对性。」

用 AskUserQuestion 问：「你有额外的竞品情报要补充吗？」
- "我有一些竞品信息想补充（产品动态、客户反馈、丢单原因等）"（真实数据路线）
- "先用预置数据练一遍"（模拟路线）

**真实数据路线**：请用户粘贴竞品信息，AI 会整合到分析中。

### Step 2：选择竞品

告诉用户：「领导让你做一份竞品分析。先确定分析对象。」

用 AskUserQuestion 问：「选择 2 个你最想分析的竞品：」（多选）
- 好分数（直接竞品：校园考试数据分析，华北为主）
- 飞象星球（直接竞品：猿辅导旗下 B 端智慧教育，AI + 内容优势）
- 七天网络（直接竞品：网上阅卷 + 基础分析，下沉市场价格战）
- 钉钉/飞书教育（相邻竞品：企业平台延伸到教育）

### Step 3：竞品地图

读取 `./references/sample-data/zhixue-competitive.md` 获取预置竞品数据。如果用户提供了额外信息，整合到分析中。

展示竞品分类地图（来自 competitive-analysis skill）：
- **直接竞品**：同样的问题、同样的用户、同样的方式 → 好分数、飞象星球、七天网络
- **相邻竞品**：目前不直接竞争但有可能进入 → 钉钉教育、飞书教育
- **替代方案**：完全不同的解决方式 → Excel 手工统计、传统教务系统

关键教学点：「最危险的竞品不一定是现在和你直接抢客户的。飞象星球从 C 端转 B 端，带着猿辅导的海量内容和互联网产品体验——这种降维打击往往比正面竞争更难防守。」

### Step 4：功能对比维度

用 AskUserQuestion 问：「功能对比矩阵需要选维度。以下哪 3 个维度最影响学校采购决策？」（多选）
- 考试数据分析深度（成绩统计、知识点诊断、进步趋势）
- 教师减负工具（自动阅卷、一键报告、智能推题）
- 教育局报表合规（数据上报、区域对比、政策指标）
- AI 个性化学习（智能推荐、自适应练习、学情预测）
- 家校沟通能力（成绩通知、学情报告分享、家长端）

### Step 5：生成功能矩阵

AI 基于用户选择的维度和竞品，生成功能对比矩阵：

```
| 能力维度 | 智学网 | [竞品1] | [竞品2] |
|---------|--------|---------|---------|
| [维度1] | ★★★★☆ | ★★★☆☆ | ★★☆☆☆ |
| [维度2] | ★★★☆☆ | ★★★★☆ | ★★★☆☆ |
| [维度3] | ★★★★★ | ★★☆☆☆ | ★★★☆☆ |
```

关键教学点：「功能矩阵要诚实。一个到处都赢的矩阵没有可信度。比如在 AI 个性化学习上，飞象星球确实领先智学网——承认这个才能让你在数据分析深度上的优势更有说服力。」

### Step 6：定位分析

AI 展示每个竞品的定位声明（Positioning Statement）：
```
对于 [目标客户] 中有 [需求] 的人，
[产品] 是一个 [品类]，
它的核心价值是 [差异化优势]。
不同于 [竞品]，[产品] 的独特之处在于 [关键差异]。
```

用 AskUserQuestion 问：「基于分析，智学网最强的定位方向是？」
- "校内评测数据最深的平台"（数据深度定位）
- "让教师一键获得学情洞察"（效率定位）
- "教育局信赖的数据合规平台"（政府资源定位）
- "从评测到学习的完整数据闭环"（平台定位）

### Step 7：战略建议

AI 基于全部分析生成 3-5 条战略建议（Strategic Implications），每条包含：
- **发现**：基于竞品分析的关键洞察
- **建议**：具体的产品策略动作
- **优先级**：高/中/低

重点关注：
- 飞象星球在 AI 个性化方面的威胁和应对策略
- 七天网络在下沉市场的价格战应对
- 智学网依托讯飞体系的独特壁垒如何强化

### Step 8：实战提示卡

展示实战提示卡：
```
以后做竞品分析时，对 AI 说：
"帮我做一份竞品分析简报。
我的产品：[产品名]，[一句话定位]
目标竞品：[竞品名1]、[竞品名2]
重点分析维度：[维度1]、[维度2]、[维度3]
我知道的竞品信息：[粘贴你掌握的情报]
请按 competitive-brief 框架输出：
竞品地图 → 功能矩阵 → 定位分析 → 战略建议"
```

[分类说明]

### [竞品1]

### [竞品2]
